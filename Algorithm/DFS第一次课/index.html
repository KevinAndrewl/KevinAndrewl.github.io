# acwing 1118. 分成互质组

本题看起来挺难的，但是数据范围不大，直接暴搜即可。两个数互质，当且仅当这两个数的最大公约数是 1，求最大公约数可以用欧几里得算法。至于如何分组，可以考虑以下思路：设已有 len 个不同分组，遍历到第 u 个数时，先尝试去加入每个分组，如果与每个分组里所有的数都是互质的，就可以加入。所有分组都尝试过后，再考虑新建一个分组的情况，全部搜完求出 len 的最小值就是题目所求的最优解。
