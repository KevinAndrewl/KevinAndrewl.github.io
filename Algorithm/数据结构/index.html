# AcWing 830. 单调栈

给定一个长度为 N 的整数数列，输出每个数左边第一个比它小的数，如果不存在则输出 − 1
数据范围：1<=N<=1051<=N<=10^51<=N<=105，数列中的元素：1<=x<=1091<=x<=10^91<=x<=109
思路：把大于aia_iai​ 的数全部 pop 掉，直到栈顶元素符合x<=aix<=a_ix<=ai​ 要求，才将数aia_iai​ 入栈
