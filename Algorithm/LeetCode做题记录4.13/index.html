# 36. 有效的数独

没有什么思路，虽然好像记得 acwing 中出现过类似的题目，似乎是用 DFS 去解决的
大佬思路使用哈希表记录数字出现的维度
关于为什么求 box 的公式为:
由于我们的 box 是从 0 开始计数的，试着想象一个 3x9 列的数组，这样的一个数组会分为 3 个 box，很明显 0/1/2 列都会属于第一个 box，后续规律即 j/3，同理可得 i/3，然后考虑到 i 大于 3 的情况，很明显如果列不变，i 的增加会多 3 个 box，所以才需要乘以 3

# 54. 螺旋矩阵

我的思考方法需要对前面走过的路径进行记录，看到一个题解不需要记录，特此记录下
思路：

  1. 首先设定上下左右边界
  2. 其次向右移动到最右，此时第一行因为已经使用过了，可以将其从图中删去，体现在代码中就是重新定义上边界
  3. 判断若重新定义后，上下边界交错，表明螺旋矩阵遍历结束，跳出循环，返回答案
  4. 若上下边界不交错，则遍历还未结束，接着向下向左向上移动，操作过程与第一，二步同理
  5. 不断循环以上步骤，直到某两条边界交错，跳出循环，返回答案
